Netflix Clone

Netflix Project

In this project I decided to recreate the Netflix homepage as I wanted to implement HTML, CSS and JavaScript into one. I initially created the basic structure with HTML then built upon it with CSS and at the end added JavaScript for interactivity. The HTML was basic as the links were not live, however the CSS allowed me to showcase not only the visuals of the site but also responsive design. I was able to make the page run smoothly on both desktops and phones alike. CSS also gave me the chance to use some hover options which made the page feel more vibrant. Once the structure and look were complete I went onto JavaScript. I was able to hide certain pieces of information which are only accessible by selecting the corresponding icons and improve upon the hover and click options.
